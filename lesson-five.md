# Positioning（定位）
>使用定位可以是CSS中重要的内容

## 相对定位
>相对于原本所在的位置，不会影响其他元素所在位置。


## 绝对定位
>与相对定位相比，不占据空间，脱离了文档流，绝对定位相对于已定位的祖先元素，如果没有祖先元素则相对于最近的包含块。

## Positioning with Floats(浮动)
>如何设置浮动

    .float-right{
  	             float: right;
                  }
    
>如何清除浮动

    .clear-float {
  	            clear: both;
                  }
